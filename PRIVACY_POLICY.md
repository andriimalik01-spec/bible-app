# Privacy Policy

This bot respects your privacy and is committed to protecting your personal data.

## 1. Data Collection
The bot may store:
- Telegram User ID
- Display name
- Language preference
- Reading progress and statistics
- Optional personal reflections (if provided)

## 2. Purpose of Data Usage
Data is used only to:
- Track reading progress
- Generate statistics and rankings
- Improve user experience

No data is sold or shared with third parties.

## 3. Data Storage
Data is stored securely on a protected server environment.

## 4. User Rights
Users may request deletion of their data at any time by contacting the administrator.

## 5. Updates
This Privacy Policy may be updated if new features are introduced.
